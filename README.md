## [LikeLion BootCamp](https://likelion.net/school) Mini Game
### 기능 구현
- **캐릭터**
  - 캐릭터 기본 이동(상,하,좌,우,점프)
  - 캐릭터 애니메이션(Idle,Run,Jump,Fall)
  - 벽 점프 및 벽 슬라이딩
  - 캐릭터 목숨 기능
    
- **팝업 기능** 
  - 점수 기록 및 랭킹 시스템
  - 로비 및 스테이지 선택

- **게임 시간**
  - 제한시간 30초, 30초 지나면 게임오버
  - 아이템 획득시 게임시간 증가
  
- **적** 
  - 적 좌우 움직임
    
- **공격**
  - 총알 공격 기능
